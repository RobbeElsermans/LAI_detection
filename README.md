# LAI_detection
Leaf Area Index detection using Digital Image Processing with Matlab
# Installation Guide for LAI_APP

This guide will walk you through the installation process for the LAI_APP from the GitHub repository.

## Steps:

1. **Navigate to the Repository:**
   - Click on the repository link provided or copy-paste it into your web browser.

2. **Access the "executable_files" Folder:**
   - Once on the repository page, locate and click on the "executable_files" folder.

3. **Find "LAI_calc" within "executable_files":**
   - Inside the "executable_files" folder, click on the "LAI_calc" folder.

4. **Open the "for_distribution" Directory:**
   - Within the "LAI_calc" folder, locate and click on the "for_distribution" folder.

5. **Download "LAI_APP.exe":**
   - Find the "LAI_APP.exe" file inside the "for_distribution" folder.
   - Click on the file to open its page.
   - Look for the "Download" button or option and save the file to your preferred location on your computer.

6. **Run the Application:**
   - Navigate to the saved location of "LAI_APP.exe."
   - Double-click on the "LAI_APP.exe" file to run the application.
![image](https://github.com/RobbeElsermans/LAI_detection/assets/46652494/ac64e1ba-ad8d-4b67-a3e6-048e1ef041c6)

7. **Follow Installation Instructions:**
   - The application will start, and you'll receive installation instructions or a setup wizard.
   - Carefully follow the on-screen instructions to complete the installation process.
